# Jarvis Desktop Assistant

A Python-based desktop assistant capable of voice interaction, system command execution, and intelligent responses.

## Features
- **Voice Interaction**: Speech-to-Text (STT) and Text-to-Speech (TTS).
- **LLM Integration**: Support for OpenAI/Gemini for intelligent conversation.
- **System Control**: Execute system commands.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Configure `.env` with your API keys (see `config/example.env`).
3. Run the assistant:
   ```bash
   python main.py
   ```
